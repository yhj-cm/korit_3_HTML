### 여는 태그 vs 닫는 태그
문법
```html
<여는태그> 내용 </닫는태그>
```
1. 여는 태그와 닫는 태그 사이의 내용은 여러 줄일 수 있는데,
그 사이에는 들여쓰기를 하는 것을 원칙으로 함.
2. 닫는 태그가 없이 단독으로 쓰이는 경우도 있다.
< 열고나서 바로 닫는 태그 형식 /> : 리액트에서는 이렇게도 사용함

1. h1 - h6 태그
2. p
    - <p>는 paragraph의 축약어로 HTML 상에서 문단을 표시할 때 사용
